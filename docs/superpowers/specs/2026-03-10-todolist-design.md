# TodoList 组件设计文档

**日期：** 2026-03-10  
**项目：** React TodoList 组件

---

## 1. 项目概述

- **类型**：React 组件
- **数据存储**：localStorage
- **目标用户**：个人日常使用
- **核心功能**：创建、编辑、完成、删除待办事项

---

## 2. UI/UX 设计

### 布局结构
```
┌─────────────────────────────┐
│        标题 + 输入框        │
├─────────────────────────────┤
│  ┌─────────────────────┐   │
│  │ ☐ 待办事项 1   [编辑][删除]│
│  ├─────────────────────┤   │
│  │ ☑ 已完成事项  [编辑][删除]│
│  └─────────────────────┘   │
└─────────────────────────────┘
```

### 组件结构
- **App 容器**：整体布局
- **Header**：标题 + 输入框
- **TodoList**：待办事项列表
- **TodoItem**：单个待办项

### 样式
- 简洁清爽，单色系
- 区分完成/未完成状态
- 响应式设计

---

## 3. 功能规格

### 数据模型
```typescript
interface Todo {
  id: string;
  text: string;
  completed: boolean;
  createdAt: number;
}
```

### 功能列表
| 功能 | 描述 |
|------|------|
| 添加待办 | 输入框回车或点击按钮添加 |
| 编辑待办 | 点击编辑按钮进入编辑模式 |
| 完成标记 | 点击 checkbox 切换状态 |
| 删除待办 | 点击删除按钮确认后删除 |
| 持久化 | localStorage 自动保存/加载 |

### 边界处理
- 空输入不添加
- 编辑时按 Enter 确认，Escape 取消
- 删除前可选择确认

---

## 4. 技术实现

- **框架**：React 18
- **样式**：CSS Modules
- **打包**：Vite（可输出独立 HTML）
- **存储**：localStorage
